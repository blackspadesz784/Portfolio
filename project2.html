Overview

An intelligent online cheating detector that monitors user behavior through the webcam to ensure exam integrity. Uses computer vision and machine learning to detect suspicious activities in real-time, including head movement, multiple persons in frame, and phone usage.

Key Features
  • Head movement tracking with MediaPipe Face Mesh
  • Multiple person detection in camera frame
  • Phone detection using object recognition AI
  • Real-time alerts and violation reporting dashboard

The Challenge

Maintaining real-time performance without lag during live sessions while running complex ML inference on the client side.

The Solution

Optimized MediaPipe landmarks processing with frame skipping and threshold smoothing algorithms for smooth 30fps inference.
